### 代码测试
下面是好几种语言的`hello, world`:

#### C
```C
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

#### CPP
```CPP
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

#### Go
```Go
package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

#### Rust
```Rust
fn main() {
    println!("Hello, World!");
}
```

#### Python
```Python
print("Hello, World!")
```
