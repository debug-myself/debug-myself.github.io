下面是好几种语言的hello, world:

### C 

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

### CPP 

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

### Go 

package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

### Rust 

fn main() {
    println!("Hello, World!");
}

### Python
